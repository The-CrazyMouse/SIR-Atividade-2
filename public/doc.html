Rotas da API

Esta documentação fornece uma visão geral das rotas disponíveis na API para a gestão de estudantes.

  • GET /students - Retorna a lista completa de estudantes registados.
  • GET /students/{id} - Retorna as informações de um estudante específico, com base no seu ID.
  • POST /students - Adiciona um novo estudante ao sistema. Campos obrigatórios: nome, idade, curso e ano.
  • PUT /students/{id} - Atualiza as informações de um estudante existente, identificado pelo ID.
  • DELETE /students/{id} - Remove um estudante do sistema, identificado pelo ID.

Para registar um novo estudante, os dados necessários incluem: nome, idade, curso e ano. Estes campos são obrigatórios para o registo de um novo estudante.
